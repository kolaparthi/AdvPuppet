# sonarCube
------------------------------------------------------------------------------------
Add the below tag in maven pom.xml to push the data to local Sonarcube server.<br>
<properties><br>
   <sonar.host.url> http://localhost:9000/ </sonar.host.url><br>
  </properties><br>
  <br>
  Default username:admin<br>
  Password:admin<br>
